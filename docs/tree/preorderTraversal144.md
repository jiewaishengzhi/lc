# LeetCode 144 - 二叉树的前序遍历

## 题目描述

给定一个二叉树的根节点 `root`，返回它的**前序遍历**结果。

**前序遍历顺序**：根 → 左 → 右

```
示例：
    1
     \
      2
     /
    3

输出：[1, 2, 3]
```

---

## 分析方法

前序遍历按「根-左-右」的顺序访问每个节点。特点是**先处理当前节点**，再递归处理左右子树。

---

## 方法一：递归

### 示例推演

```
树：
        4
       / \
      2   6
     / \ / \
    1  3 5  7

══════════════════════════════════════════════════════════════

dfs(节点4)
  ├── 访问4 → result.add(4)  ← 第1个
  ├── dfs(节点2)
  │     ├── 访问2 → result.add(2)  ← 第2个
  │     ├── dfs(节点1)
  │     │     ├── 访问1 → result.add(1)  ← 第3个
  │     │     ├── dfs(null) → 返回
  │     │     └── dfs(null) → 返回
  │     └── dfs(节点3)
  │           ├── 访问3 → result.add(3)  ← 第4个
  │           ├── dfs(null) → 返回
  │           └── dfs(null) → 返回
  └── dfs(节点6)
        ├── 访问6 → result.add(6)  ← 第5个
        ├── dfs(节点5)
        │     ├── 访问5 → result.add(5)  ← 第6个
        │     ├── dfs(null) → 返回
        │     └── dfs(null) → 返回
        └── dfs(节点7)
              ├── 访问7 → result.add(7)  ← 第7个
              ├── dfs(null) → 返回
              └── dfs(null) → 返回

结果：[4, 2, 1, 3, 6, 5, 7]

访问顺序：
        4
       / \
      2   6       4→2→1→3→6→5→7
     / \ / \      ↑根 ←左子树→ ←右子树→
    1  3 5  7
```

### 代码

```java
public List<Integer> result = new ArrayList<>();

public List<Integer> preorderTraversal(TreeNode root) {
    dfs(root);
    return result;
}

private void dfs(TreeNode node) {
    if (node == null) return;
    result.add(node.val);   // 根（先处理当前节点）
    dfs(node.left);          // 左
    dfs(node.right);         // 右
}
```

### 核心流程图

```
         ┌──────────────┐
         │  dfs(node)   │
         └──────┬───────┘
                │
                ▼
        ┌──────────────┐     Yes
        │ node==null?  │────────→ return
        └──────┬───────┘
               │ No
               ▼
     ┌──────────────────┐
     │ result.add(val)  │  ← 先处理当前节点（根）
     └────────┬─────────┘
              │
              ▼
     ┌──────────────────┐
     │ dfs(node.left)   │  ← 再左子树
     └────────┬─────────┘
              │
              ▼
     ┌──────────────────┐
     │ dfs(node.right)  │  ← 最后右子树
     └──────────────────┘
```

## 方法二：迭代

### 分析思路

用栈模拟递归。前序遍历先处理当前节点，所以弹出时就处理。

关键：**右孩子先入栈，左孩子后入栈**（栈后进先出，左后入栈先弹出）。

### 示例推演

```
树：
        4
       / \
      2   6
     / \ / \
    1  3 5  7

══════════════════════════════════════════════════════════════

初始：栈 = [4]

──────────────────────────────────────────────────────────────
弹出 4 → result.add(4)  ← 第1个
右6先入栈，左2后入栈 → 栈：[6, 2]
──────────────────────────────────────────────────────────────
弹出 2 → result.add(2)  ← 第2个
右3先入栈，左1后入栈 → 栈：[6, 3, 1]
──────────────────────────────────────────────────────────────
弹出 1 → result.add(1)  ← 第3个
无孩子 → 栈：[6, 3]
──────────────────────────────────────────────────────────────
弹出 3 → result.add(3)  ← 第4个
无孩子 → 栈：[6]
──────────────────────────────────────────────────────────────
弹出 6 → result.add(6)  ← 第5个
右7先入栈，左5后入栈 → 栈：[7, 5]
──────────────────────────────────────────────────────────────
弹出 5 → result.add(5)  ← 第6个
无孩子 → 栈：[7]
──────────────────────────────────────────────────────────────
弹出 7 → result.add(7)  ← 第7个
无孩子 → 栈：[]

══════════════════════════════════════════════════════════════
【过程汇总】
══════════════════════════════════════════════════════════════

弹出  result              栈变化
──────────────────────────────────────────────
4     [4]                 [6,2]
2     [4,2]               [6,3,1]
1     [4,2,1]             [6,3]
3     [4,2,1,3]           [6]
6     [4,2,1,3,6]         [7,5]
5     [4,2,1,3,6,5]       [7]
7     [4,2,1,3,6,5,7]     []

结果：[4, 2, 1, 3, 6, 5, 7] ✅
```

### 代码

```java
public List<Integer> preorderTraversal2(TreeNode root) {
    List<Integer> result = new ArrayList<>();
    if (root == null) return result;

    Deque<TreeNode> stack = new ArrayDeque<>();
    stack.push(root);

    while (!stack.isEmpty()) {
        TreeNode node = stack.pop();
        result.add(node.val);       // 处理当前节点

        // 右孩子先入栈（后弹出）
        if (node.right != null) stack.push(node.right);
        // 左孩子后入栈（先弹出）
        if (node.left != null) stack.push(node.left);
    }
    return result;
}
```

### 核心流程图

```
┌──────────────────────────────────────────────────────────┐
│  stack.push(root)                                        │
└──────────────────────────────────────────────────────────┘
                         │
                         ▼
               ┌──────────────────┐
               │    栈空?         │◄──────────────┐
               └───────┬──────────┘               │
                   Yes │  No                      │
                   │   │                          │
                   ▼   ▼                          │
              ┌──────┐ ┌──────────────────┐       │
              │return│ │ node = pop()     │       │
              │result│ │ result.add(val)  │       │
              └──────┘ └────────┬─────────┘       │
                                │                 │
                                ▼                 │
                     ┌──────────────────────┐     │
                     │ right != null?       │     │
                     │ → push(right) 先入栈 │     │
                     └──────────┬───────────┘     │
                                │                 │
                                ▼                 │
                     ┌──────────────────────┐     │
                     │ left != null?        │     │
                     │ → push(left) 后入栈  │─────┘
                     └──────────────────────┘
```

## 两种方法对比

| 对比项 | 递归 | 迭代 |
|--------|------|------|
| 代码量 | 少 | 稍多 |
| 入栈顺序 | 自然递归 | **右先左后** |
| 空间 | O(h) 递归栈 | O(h) 显式栈 |
| 栈溢出风险 | 有 | 无 |

---

## 与中序遍历（94）迭代的区别

| 对比项 | 前序（144） | 中序（94） |
|--------|------------|------------|
| 处理时机 | **弹出时**就处理 | 压栈到底再弹出处理 |
| 入栈方式 | 弹一个压两个（右先左后） | 一路向左压栈 |
| 循环结构 | 一个 while | 两个 while（外层+内层向左） |

---

## 复杂度分析

| 复杂度 | 值 | 说明 |
|--------|-----|------|
| 时间 | O(n) | 每个节点访问一次 |
| 空间 | O(h) | 栈深度 = 树高度 |

---

## 关键点总结

1. **前序 = 根→左→右**：先处理当前节点，再处理左右子树
2. **迭代时右先入栈**：栈后进先出，左后入栈才能先弹出
3. **和中序迭代的区别**：前序弹出时就处理，中序是压栈到底再处理
4. **应用场景**：LeetCode 297 序列化、105 构建树都用前序
