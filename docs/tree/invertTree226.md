# LeetCode 226 - 翻转二叉树

## 题目描述

给你一棵二叉树的根节点 `root`，翻转这棵二叉树，并返回其根节点。

**翻转**：将每个节点的左右子树互换。

```
示例：

      4                    4
    /   \                /   \
   2     7      →       7     2
  / \   / \            / \   / \
 1   3 6   9          9   6 3   1

```

---

## 分析方法

翻转二叉树的本质是对每个节点执行「交换左右孩子」操作。

可用两种方法实现：
1. **递归（DFS）**：自顶向下或自底向上翻转
2. **迭代（BFS）**：层序遍历，逐层交换

---

## 方法一：递归

### 分析思路

1. **终止条件**：当前节点为空，返回 null
2. **当前层操作**：交换左右孩子
3. **递归处理**：对左右子树执行相同操作

### 示例推演

### 代码实现

```java
public TreeNode invertTree(TreeNode root) {
    // 1. 递归终止条件：当前节点为空
    if (root == null) {
        return null;
    }

    // 2. 交换当前节点的左右孩子
    TreeNode temp = root.left;
    root.left = root.right;
    root.right = temp;

    // 3. 递归翻转左右子树
    invertTree(root.left);
    invertTree(root.right);

    return root;
}
```

## 方法二：迭代（BFS）

### 分析思路

使用队列进行层序遍历，对每个出队的节点交换其左右孩子，然后将孩子入队继续处理。

### 示例推演

```
原始树：
      4
    /   \
   2     7
  / \   / \
 1   3 6   9

══════════════════════════════════════════════════════════════
【迭代过程】
══════════════════════════════════════════════════════════════

初始状态：
queue = [4]

──────────────────────────────────────────────────────────────
【第1轮】
──────────────────────────────────────────────────────────────

弹出节点 4，交换左右孩子 (2 ↔ 7)

树变为：
      4
    /   \
   7     2    ← 已交换
  / \   / \
 6   9 1   3

将 7, 2 入队
queue = [7, 2]

──────────────────────────────────────────────────────────────
【第2轮】
──────────────────────────────────────────────────────────────

弹出节点 7，交换左右孩子 (6 ↔ 9)

树变为：
      4
    /   \
   7     2
  / \   / \
 9   6 1   3    ← 6和9已交换

将 9, 6 入队
queue = [2, 9, 6]

──────────────────────────────────────────────────────────────
【第3轮】
──────────────────────────────────────────────────────────────

弹出节点 2，交换左右孩子 (1 ↔ 3)

树变为：
      4
    /   \
   7     2
  / \   / \
 9   6 3   1    ← 1和3已交换

将 3, 1 入队
queue = [9, 6, 3, 1]

──────────────────────────────────────────────────────────────
【第4-7轮】
──────────────────────────────────────────────────────────────

依次弹出 9, 6, 3, 1，它们都没有孩子，不产生新入队

queue = []

══════════════════════════════════════════════════════════════
【最终结果】
══════════════════════════════════════════════════════════════

      4
    /   \
   7     2
  / \   / \
 9   6 3   1

迭代轮数：7轮（处理7个节点）
```

### 代码实现

```java
public TreeNode invertTree2(TreeNode root) {
    if (root == null) return null;

    Queue<TreeNode> queue = new LinkedList<>();
    queue.offer(root);

    while (!queue.isEmpty()) {
        TreeNode node = queue.poll();

        // 交换左右孩子
        TreeNode temp = node.left;
        node.left = node.right;
        node.right = temp;

        // 将非空孩子入队
        if (node.left != null) {
            queue.offer(node.left);
        }
        if (node.right != null) {
            queue.offer(node.right);
        }
    }
    return root;
}
```

## 两种方法对比

| 对比项 | 递归 | 迭代 |
|--------|------|------|
| 空间复杂度 | O(h) 递归栈 | O(w) 队列宽度 |
| 代码简洁度 | 更简洁 | 稍复杂 |
| 栈溢出风险 | 有（树很深时） | 无 |
| 遍历方式 | DFS | BFS |

> h = 树高度，w = 树最大宽度

---

## 复杂度分析

| 复杂度 | 值 | 说明 |
|--------|-----|------|
| 时间 | O(n) | 每个节点访问一次 |
| 空间 | O(n) | 递归栈/队列最坏情况 |

---

## 关键点总结

1. **先交换再递归**：也可以先递归再交换，效果相同
2. **注意空节点**：递归和迭代都要处理
3. **原地修改**：直接在原树上交换，不需要创建新节点
