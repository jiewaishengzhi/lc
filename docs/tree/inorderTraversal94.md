# LeetCode 94 - 二叉树的中序遍历

## 题目描述

给定一个二叉树的根节点 `root`，返回它的**中序遍历**结果。

**中序遍历顺序**：左 → 根 → 右

```
示例：
    1
     \
      2
     /
    3

输出：[1, 3, 2]
```

---

## 分析方法

中序遍历按「左-根-右」的顺序访问每个节点。对于 BST，中序遍历的结果一定是递增序列。

---

## 方法一：递归

### 示例推演

```
树：
        4
       / \
      2   6
     / \ / \
    1  3 5  7

══════════════════════════════════════════════════════════════

dfs(节点4)
  ├── dfs(节点2)
  │     ├── dfs(节点1)
  │     │     ├── dfs(null) → 返回
  │     │     ├── 访问1 → result.add(1)  ← 第1个
  │     │     └── dfs(null) → 返回
  │     │
  │     ├── 访问2 → result.add(2)        ← 第2个
  │     │
  │     └── dfs(节点3)
  │           ├── dfs(null) → 返回
  │           ├── 访问3 → result.add(3)  ← 第3个
  │           └── dfs(null) → 返回
  │
  ├── 访问4 → result.add(4)              ← 第4个
  │
  └── dfs(节点6)
        ├── dfs(节点5)
        │     ├── dfs(null) → 返回
        │     ├── 访问5 → result.add(5)  ← 第5个
        │     └── dfs(null) → 返回
        │
        ├── 访问6 → result.add(6)        ← 第6个
        │
        └── dfs(节点7)
              ├── dfs(null) → 返回
              ├── 访问7 → result.add(7)  ← 第7个
              └── dfs(null) → 返回

结果：[1, 2, 3, 4, 5, 6, 7]

访问顺序：
        4
       / \
      2   6       1→2→3→4→5→6→7
     / \ / \      ←左子树→ ↑ ←右子树→
    1  3 5  7             根
```

### 代码

```java
private List<Integer> result = new ArrayList<>();

public List<Integer> inorderTraversal(TreeNode root) {
    dfs(root);
    return result;
}

private void dfs(TreeNode node) {
    if (node == null) return;
    dfs(node.left);            // 左
    result.add(node.val);      // 根
    dfs(node.right);           // 右
}
```

### 核心流程图

```
         ┌──────────────┐
         │  dfs(node)   │
         └──────┬───────┘
                │
                ▼
        ┌──────────────┐     Yes
        │ node==null?  │────────→ return
        └──────┬───────┘
               │ No
               ▼
     ┌──────────────────┐
     │ dfs(node.left)   │  ← 先左子树
     └────────┬─────────┘
              │
              ▼
     ┌──────────────────┐
     │ result.add(val)  │  ← 再处理当前节点
     └────────┬─────────┘
              │
              ▼
     ┌──────────────────┐
     │ dfs(node.right)  │  ← 最后右子树
     └──────────────────┘
```

## 方法二：迭代

### 分析思路

用栈模拟递归：
1. 一路向左压栈（模拟递归左子树）
2. 弹出栈顶处理（模拟返回到当前节点）
3. 转向右子树（模拟递归右子树）

### 示例推演

```
树：
        4
       / \
      2   6
     / \ / \
    1  3 5  7

══════════════════════════════════════════════════════════════

操作          弹出   result              栈状态
─────────────────────────────────────────────────────────
压栈4,2,1     -      []                  [4,2,1]
弹1           1      [1]                 [4,2]
弹2           2      [1,2]               [4]
压栈3         -      [1,2]               [4,3]
弹3           3      [1,2,3]             [4]
弹4           4      [1,2,3,4]           []
压栈6,5       -      [1,2,3,4]           [6,5]
弹5           5      [1,2,3,4,5]         [6]
弹6           6      [1,2,3,4,5,6]       []
压栈7         -      [1,2,3,4,5,6]       [7]
弹7           7      [1,2,3,4,5,6,7]     []

结果：[1, 2, 3, 4, 5, 6, 7] ✅
```

### 代码

```java
public List<Integer> inorderTraversal2(TreeNode root) {
    List<Integer> result = new ArrayList<>();
    Deque<TreeNode> stack = new ArrayDeque<>();

    while (root != null || !stack.isEmpty()) {
        // 一路向左压栈
        while (root != null) {
            stack.push(root);
            root = root.left;
        }

        // 弹出栈顶，处理
        root = stack.pop();
        result.add(root.val);

        // 转向右子树
        root = root.right;
    }
    return result;
}
```

### 核心流程图

```
┌──────────────────────────────────────────────┐
│        root != null 或 栈非空?                │◄─────────┐
└──────────────────┬───────────────────────────┘          │
       No          │          Yes                          │
        │          │                                       │
        ▼          ▼                                       │
  ┌──────────┐  ┌──────────────────────┐                  │
  │return    │  │ 一路向左压栈          │                  │
  │result    │  │ while(root!=null)     │                  │
  └──────────┘  │   stack.push(root)    │                  │
                │   root = root.left    │                  │
                └──────────┬────────────┘                  │
                           │                               │
                           ▼                               │
                ┌──────────────────────┐                   │
                │ root = stack.pop()   │                   │
                │ result.add(root.val) │  ← 处理当前节点    │
                └──────────┬────────────┘                  │
                           │                               │
                           ▼                               │
                ┌──────────────────────┐                   │
                │ root = root.right    │  ← 转向右子树      │
                └──────────────────────┘───────────────────┘
```

## 两种方法对比

| 对比项 | 递归 | 迭代 |
|--------|------|------|
| 代码量 | 少 | 稍多 |
| 理解难度 | 直观 | 需理解栈模拟 |
| 空间 | O(h) 递归栈 | O(h) 显式栈 |
| 栈溢出风险 | 有（树很深时） | 无 |

---

## 复杂度分析

| 复杂度 | 值 | 说明 |
|--------|-----|------|
| 时间 | O(n) | 每个节点访问一次 |
| 空间 | O(h) | 栈深度 = 树高度 |

---

## 关键点总结

1. **左→根→右**：中序遍历 BST 得到递增序列
2. **迭代的两个 while**：外层控制整体流程，内层负责向左走到底
3. **栈的作用**：保存回溯点，左走到底后能回到上层节点
4. **和前序迭代的区别**：中序是压栈到底再弹出处理，前序是弹出时处理再压栈
