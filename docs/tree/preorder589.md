# LeetCode 589 - N 叉树的前序遍历

## 题目描述

给定一个 N 叉树的根节点 `root`，返回其节点值的**前序遍历**。

N 叉树的每个节点可以有多个子节点，子节点存储在 `children` 列表中。

```
示例：
        1
      / | \
     3  2  4
    / \
   5   6

输出：[1, 3, 5, 6, 2, 4]
```

**前序遍历顺序**：先访问根节点，再从左到右依次遍历每个子节点。

---

## 分析方法

与二叉树前序遍历本质相同，区别在于子节点从固定的 2 个（left/right）变成了不固定的 N 个（children 列表）。

### 节点结构

```java
class Node {
    public int val;
    public List<Node> children; // 子节点列表
}
```

---

## 方法一：递归

### 示例推演

```
树：
        1
      / | \
     3  2  4
    / \
   5   6

══════════════════════════════════════════════════════════════

dfs(节点1)
  ├── 访问1 → result.add(1)  ← 第1个
  ├── 遍历 children: [3, 2, 4]
  │
  ├── dfs(节点3)
  │     ├── 访问3 → result.add(3)  ← 第2个
  │     ├── 遍历 children: [5, 6]
  │     ├── dfs(节点5)
  │     │     ├── 访问5 → result.add(5)  ← 第3个
  │     │     └── children 为空 → 返回
  │     └── dfs(节点6)
  │           ├── 访问6 → result.add(6)  ← 第4个
  │           └── children 为空 → 返回
  │
  ├── dfs(节点2)
  │     ├── 访问2 → result.add(2)  ← 第5个
  │     └── children 为空 → 返回
  │
  └── dfs(节点4)
        ├── 访问4 → result.add(4)  ← 第6个
        └── children 为空 → 返回

结果：[1, 3, 5, 6, 2, 4] ✅

访问顺序：
        1
      / | \
     3  2  4     1→3→5→6→2→4
    / \           ↑ 根  ←3的子树→ ←其余→
   5   6
```

### 代码

```java
private List<Integer> result = new ArrayList<>();

public List<Integer> preorder(Node root) {
    dfs(root);
    return result;
}

private void dfs(Node node) {
    if (node == null) return;

    result.add(node.val);                // 先访问当前节点
    for (Node child : node.children) {   // 从左到右遍历每个子节点
        dfs(child);
    }
}
```

### 核心流程图

```
         ┌──────────────┐
         │  dfs(node)   │
         └──────┬───────┘
                │
                ▼
        ┌──────────────┐     Yes
        │ node==null?  │────────→ return
        └──────┬───────┘
               │ No
               ▼
     ┌──────────────────┐
     │ result.add(val)  │  ← 先处理当前节点
     └────────┬─────────┘
              │
              ▼
     ┌──────────────────────────┐
     │ for child in children:   │◄──────┐
     │   dfs(child)             │       │
     └──────────┬───────────────┘       │
                │                       │
                ▼                       │
        ┌──────────────┐         Yes    │
        │ 还有下一个   │───────────────┘
        │ child?       │
        └──────┬───────┘
            No │
               ▼
            return
```

## 方法二：迭代

### 分析思路

和二叉树前序迭代思路类似，但子节点有多个，需要**从右到左**入栈（保证从左到右弹出）。

```
二叉树：右孩子先入栈，左孩子后入栈 → 左先弹出
N叉树：children 从右到左入栈 → 左边的孩子先弹出
```

### 示例推演

```
树：
        1
      / | \
     3  2  4
    / \
   5   6

══════════════════════════════════════════════════════════════

初始：栈 = [1]

──────────────────────────────────────────────────────────────
弹出 1 → result.add(1)  ← 第1个
children=[3,2,4]，从右到左入栈：push(4), push(2), push(3)
栈：[4, 2, 3]
──────────────────────────────────────────────────────────────
弹出 3 → result.add(3)  ← 第2个
children=[5,6]，从右到左入栈：push(6), push(5)
栈：[4, 2, 6, 5]
──────────────────────────────────────────────────────────────
弹出 5 → result.add(5)  ← 第3个
children=[]，不入栈
栈：[4, 2, 6]
──────────────────────────────────────────────────────────────
弹出 6 → result.add(6)  ← 第4个
children=[]，不入栈
栈：[4, 2]
──────────────────────────────────────────────────────────────
弹出 2 → result.add(2)  ← 第5个
children=[]，不入栈
栈：[4]
──────────────────────────────────────────────────────────────
弹出 4 → result.add(4)  ← 第6个
children=[]，不入栈
栈：[]
──────────────────────────────────────────────────────────────

══════════════════════════════════════════════════════════════
【过程汇总】
══════════════════════════════════════════════════════════════

弹出  result              入栈顺序        栈状态
─────────────────────────────────────────────────────
1     [1]                 4→2→3(右到左)   [4,2,3]
3     [1,3]               6→5(右到左)     [4,2,6,5]
5     [1,3,5]             无              [4,2,6]
6     [1,3,5,6]           无              [4,2]
2     [1,3,5,6,2]         无              [4]
4     [1,3,5,6,2,4]       无              []

结果：[1, 3, 5, 6, 2, 4] ✅
```

### 代码

```java
public List<Integer> preorder2(Node root) {
    List<Integer> result = new ArrayList<>();
    if (root == null) return result;

    Deque<Node> stack = new ArrayDeque<>();
    stack.push(root);

    while (!stack.isEmpty()) {
        Node node = stack.pop();
        result.add(node.val);               // 处理当前节点

        // 从右到左入栈，保证左边孩子先被弹出
        for (int i = node.children.size() - 1; i >= 0; i--) {
            stack.push(node.children.get(i));
        }
    }
    return result;
}
```

### 核心流程图

```
┌──────────────────────────────────────────────────────────┐
│  stack.push(root)                                        │
└──────────────────────────────────────────────────────────┘
                         │
                         ▼
               ┌──────────────────┐
               │    栈空?         │◄──────────────┐
               └───────┬──────────┘               │
                   Yes │  No                      │
                   │   │                          │
                   ▼   ▼                          │
              ┌──────┐ ┌──────────────────┐       │
              │return│ │ node = pop()     │       │
              │result│ │ result.add(val)  │       │
              └──────┘ └────────┬─────────┘       │
                                │                 │
                                ▼                 │
                  ┌──────────────────────────┐    │
                  │ children 从右到左入栈     │    │
                  │ for i=n-1 → 0:           │    │
                  │   stack.push(child[i])   │────┘
                  └──────────────────────────┘
```

## 两种方法对比

| 对比项 | 递归 | 迭代 |
|--------|------|------|
| 代码量 | 少 | 稍多 |
| 入栈顺序 | 自然递归 | children 从右到左 |
| 空间 | O(n) 递归栈 | O(n) 显式栈 |
| 栈溢出风险 | 有 | 无 |

---

## 复杂度分析

| 复杂度 | 值 | 说明 |
|--------|-----|------|
| 时间 | O(n) | 每个节点访问一次 |
| 空间 | O(n) | 栈/递归栈 |

---

## 关键点总结

1. **前序 = 先根再子节点**：和二叉树前序本质一样
2. **迭代时 children 从右到左入栈**：保证从左到右弹出
3. **递归用 for 循环遍历 children**：替代二叉树的两个 dfs 调用
4. **注意 children 可能为空**：叶子节点没有子节点
5. **与二叉树前序（144）的区别**：子节点从 2 个变成 N 个，遍历方式用 for 循环替代两次 dfs
