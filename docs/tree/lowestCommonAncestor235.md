# LeetCode 235 - 二叉搜索树的最近公共祖先

## 题目描述

给定一个 BST 和两个节点 `p`、`q`，找到它们的**最近公共祖先（LCA）**。

**公共祖先**：一个节点同时是 p 和 q 的祖先（一个节点也可以是它自己的祖先）。

**最近**：离 p、q 最近的那个祖先。

```
示例：
        6
       / \
      2   8
     / \ / \
    0  4 7  9
      / \
     3   5

p = 2, q = 8 → LCA = 6
p = 2, q = 4 → LCA = 2（2本身就是4的祖先）
```

---

## 分析方法

利用 BST 性质：**左 < 根 < 右**

| 情况 | 条件 | 操作 |
|------|------|------|
| p, q 都在右侧 | `p.val > cur.val` 且 `q.val > cur.val` | 往右走 |
| p, q 都在左侧 | `p.val < cur.val` 且 `q.val < cur.val` | 往左走 |
| 一左一右 或 命中 p/q | — | **当前节点就是 LCA** |

---

## 方法一：递归

### 示例推演

```
树：
        6
       / \
      2   8
     / \ / \
    0  4 7  9
      / \
     3   5

══════════════════════════════════════════════════════════════
【示例1】p = 2, q = 8
══════════════════════════════════════════════════════════════

lowestCommonAncestor(节点6, p=2, q=8)
  p.val=2 < 6 且 q.val=8 > 6
  → 一左一右 ✅ → return 节点6

LCA = 6

══════════════════════════════════════════════════════════════
【示例2】p = 2, q = 4
══════════════════════════════════════════════════════════════

lowestCommonAncestor(节点6, p=2, q=4)
  p.val=2 < 6 且 q.val=4 < 6
  → 都在左侧 → 递归 lowestCommonAncestor(节点2, p=2, q=4)

lowestCommonAncestor(节点2, p=2, q=4)
  p.val=2 == 2 ✅
  → 不满足"都>cur"也不满足"都<cur"
  → return 节点2

LCA = 2

══════════════════════════════════════════════════════════════
【示例3】p = 3, q = 5
══════════════════════════════════════════════════════════════

lowestCommonAncestor(节点6, p=3, q=5)
  3<6 且 5<6 → 都在左 → 递归节点2

lowestCommonAncestor(节点2, p=3, q=5)
  3>2 且 5>2 → 都在右 → 递归节点4

lowestCommonAncestor(节点4, p=3, q=5)
  3<4 且 5>4 → 一左一右 ✅ → return 节点4

LCA = 4

══════════════════════════════════════════════════════════════
【else 分支的4种具体情形】
══════════════════════════════════════════════════════════════

情形1: p在左, q在右        情形2: p在右, q在左
        cur                       cur
       /   \                     /   \
      p     q                   q     p

情形3: cur就是p              情形4: cur就是q
        cur(p)                     cur(q)
       /   \                     /   \
      q   (或右)              (或左)   p

共同点：cur 是分叉点 或 cur 本身就是 p/q → cur 就是 LCA
```

### 代码

```java
public TreeNode lowestCommonAncestor(TreeNode root, TreeNode p, TreeNode q) {
    // 防御性编程
    if (root == null) return null;

    // p, q 都在右子树
    if (p.val > root.val && q.val > root.val) {
        return lowestCommonAncestor(root.right, p, q);
    }
    // p, q 都在左子树
    if (p.val < root.val && q.val < root.val) {
        return lowestCommonAncestor(root.left, p, q);
    }
    // 一左一右 或 其中一个等于root → 当前就是LCA
    return root;
}
```

### 核心流程图

```
         ┌──────────────────────────┐
         │  lowestCommonAncestor    │
         │  (root, p, q)           │
         └────────────┬─────────────┘
                      │
                      ▼
            ┌──────────────────┐
            │ p,q 都 > root?  │──Yes──→ 递归 root.right
            └────────┬─────────┘
                     │ No
                     ▼
            ┌──────────────────┐
            │ p,q 都 < root?  │──Yes──→ 递归 root.left
            └────────┬─────────┘
                     │ No
                     ▼
            ┌──────────────────┐
            │ return root      │  ← 分叉点/LCA
            └──────────────────┘
```

## 方法二：迭代

### 示例推演

```
══════════════════════════════════════════════════════════════
【p = 3, q = 5】
══════════════════════════════════════════════════════════════

第1轮: curr=6,  3<6, 5<6  → 都在左 → curr=2
第2轮: curr=2,  3>2, 5>2  → 都在右 → curr=4
第3轮: curr=4,  3<4, 5>4  → 一左一右 → return 4

LCA = 4（只用了3次循环，空间O(1)）
```

### 代码

```java
public TreeNode lowestCommonAncestor2(TreeNode root, TreeNode p, TreeNode q) {
    TreeNode curr = root;
    while (curr != null) {
        if (p.val > curr.val && q.val > curr.val) {
            curr = curr.right;       // 都在右
        } else if (p.val < curr.val && q.val < curr.val) {
            curr = curr.left;        // 都在左
        } else {
            return curr;             // 分叉点，就是LCA
        }
    }
    return null;
}
```

### 核心流程图

```
┌──────────────────────────────────────────────────────────┐
│           curr = root                                    │
└──────────────────────────────────────────────────────────┘
                         │
                         ▼
               ┌──────────────────┐
               │   curr != null?  │◄──────────────┐
               └───────┬──────────┘               │
                   No  │  Yes                      │
                    ▼  │                           │
              return null▼                          │
            ┌────────────────────┐                  │
            │ p,q 都 > curr.val? │──Yes──→ curr=right┘
            └────────┬───────────┘
                     │ No
                     ▼
            ┌────────────────────┐
            │ p,q 都 < curr.val? │──Yes──→ curr=left──┘
            └────────┬───────────┘
                     │ No
                     ▼
            ┌──────────────────┐
            │   return curr    │  ← LCA
            └──────────────────┘
```

## 两种方法对比

| 对比项 | 递归 | 迭代 |
|--------|------|------|
| 空间 | O(h) 递归栈 | **O(1)** |
| 代码 | 简洁 | 同样简洁 |
| 性能 | 有栈开销 | 更优 |

---

## 复杂度分析

| 复杂度 | 递归 | 迭代 |
|--------|------|------|
| 时间 | O(h) | O(h) |
| 空间 | O(h) | O(1) |

> h = 树高，平衡树 O(log n)，链状树 O(n)

---

## 关键点总结

1. **BST 性质是关键**：左 < 根 < 右，一次比较就能判断方向
2. **遇到分叉点就是 LCA**：p、q 不在同一子树时，当前节点即答案
3. **节点本身可以是 LCA**：如 p=2, q=4 → LCA=2
4. **else 的4种情形**：p左q右、p右q左、cur==p、cur==q
