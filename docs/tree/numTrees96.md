# LeetCode 96 - 不同的二叉搜索树

## 题目描述

给你一个整数 `n`，求恰由 `n` 个节点组成且节点值从 `1` 到 `n` 互不相同的 BST 的种数。

```
n = 3 → 输出 5

5种BST：
  1        1         2        3       3
   \        \       / \      /       /
    2        3     1   3    1       2
     \      /               \     /
      3    2                 2   1
```

---

## 分析方法

### 核心思路：枚举根节点

对于 `1~n` 个节点，每个节点都可以做根：

```
以 i 为根节点（1 ≤ i ≤ n）：
  左子树：1 ~ i-1（共 i-1 个节点）
  右子树：i+1 ~ n（共 n-i 个节点）

左子树的种数 × 右子树的种数 = 以 i 为根的总种数
```

**具体数值不影响种数，只看节点个数**。

### 定义

设 `G(n)` = n 个节点能组成的 BST 种数

```
G(n) = G(0)×G(n-1) + G(1)×G(n-2) + ... + G(n-1)×G(0)
```

这就是**卡特兰数（Catalan Number）**。

### 手动跑一遍 n=3 的代码

```
G = [1, 1, 0, 0]

i=2:
  j=1: G[2] += G[0]*G[1] = 1*1 = 1 → G[2]=1
  j=2: G[2] += G[1]*G[0] = 1*1 = 1 → G[2]=2
G = [1, 1, 2, 0]

i=3:
  j=1: G[3] += G[0]*G[2] = 1*2 = 2 → G[3]=2
  j=2: G[3] += G[1]*G[1] = 1*1 = 1 → G[3]=3
  j=3: G[3] += G[2]*G[0] = 2*1 = 2 → G[3]=5
G = [1, 1, 2, 5]

return G[3] = 5 ✅
```

## 代码实现

```java
public int numTrees(int n) {
    int[] G = new int[n + 1];
    G[0] = 1;  // 空树1种
    G[1] = 1;  // 1个节点1种

    // 依次计算 G(2), G(3), ..., G(n)
    for (int i = 2; i <= n; i++) {
        // 枚举每个节点 j 作为根
        for (int j = 1; j <= i; j++) {
            G[i] += G[j - 1] * G[i - j];
            //      左子树种数 × 右子树种数
        }
    }
    return G[n];
}
```

## 关键点总结

1. **核心公式**：`G(n) = Σ G(j-1) × G(n-j)`，枚举每个根
2. **G(0) = 1**：空树也是一种合法 BST，是递推的基础
3. **与具体数值无关**：只看节点个数，不看具体值
4. **卡特兰数**：G(n) 就是第 n 个卡特兰数
5. **动态规划**：从小到大依次计算，利用已算好的小规模结果
